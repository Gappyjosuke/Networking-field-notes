# Networking & Cybersecurity Journal

>[!NOTE]
> A place where I document random things I learn while surfing the internet, reading Reddit threads, experimenting with Linux, using Wireshark, debugging weird network behavior, or simply going down random technical rabbit holes.
>
> Sometimes I start learning one small thing and then end up spending hours trying to understand how everything underneath actually works.
>
> So this repo is basically me documenting:
>
> - things I learned
> - commands I used
> - packet captures
> - weird behavior I noticed
> - what I thought was happening
> - wrong assumptions I made
> - random theories/questions that came into my mind
> - how certain protocols or attacks might work
> - what was actually happening after digging deeper
>
> Most of the notes here are written more like investigation logs or personal learning trails rather than polished tutorials.
